# PowerShell Scripts

## Overview

This folder contains PowerShell scripts created while learning and practicing PowerShell administration, automation, and scripting concepts.

The goal of these projects is to apply concepts learned through hands-on exercises, labs, and real-world problem solving. Scripts in this folder are developed incrementally as new PowerShell skills are learned and practiced.

## Purpose

These scripts are used to:

* Reinforce PowerShell concepts learned through study and lab exercises
* Practice automation and administrative tasks
* Develop troubleshooting and scripting skills
* Build a portfolio of PowerShell projects
* Demonstrate growth and progression as a PowerShell practitioner

## Topics Covered

Projects in this folder may include:

* System administration
* Process and service management
* File and folder management
* Reporting and auditing
* Data filtering and sorting
* Automation workflows
* Security and compliance checks
* PowerShell object manipulation
* Pipeline processing
* Remote administration (future projects)

## Current Projects

| Project            | Description                                                                                                      |
| ------------------ | ---------------------------------------------------------------------------------------------------------------- |
| Endpoint Snapshot  | Read-only endpoint assessment tool that gathers system, user, process, service, and storage information to provide a quick security and operational overview of a Windows device. Built as a PowerShell learning project to practice endpoint auditing, object manipulation, filtering, formatting, and reporting concepts. |

## Notes

These projects are intended as learning exercises and may evolve over time as new PowerShell concepts and best practices are learned.

Scripts may be updated, expanded, or refactored as additional knowledge and experience are gained.
