# PowerShell-Learning

Repository for hands-on PowerShell learning based on the book *Learn PowerShell in a Month of Lunches*.

---

# PowerShell Learning Overview

This repository is a dedicated workspace for learning and practicing PowerShell through:
- Chapter labs
- Command discovery exercises
- Troubleshooting workflows
- Scripting practice
- Security-focused experimentation
- Automation concepts

The goal of this repository is to document practical progress while developing PowerShell, scripting, and cybersecurity-related technical skills.

---

# Purpose

This repository serves as:
- A structured PowerShell learning environment
- A scripting practice workspace
- A cybersecurity automation lab
- A portfolio of PowerShell progress and projects
- A documentation space for troubleshooting and experimentation

---

# Topics Covered

Current topics include:
- PowerShell help system usage
- Wildcard searches and command discovery
- Aliases and abbreviated commands
- Process and service management
- Transcript logging
- Out-File parameters and file handling
- Object-oriented cmdlets
- PowerShell help topics (`about_*`)
- Parameter interpretation and syntax analysis
- Basic PowerShell troubleshooting workflows

Future topics may include:
- Variables and arrays
- Pipelines and formatting
- Functions and modules
- Error handling
- File and system management
- Network and port auditing
- Windows administration
- Security and system auditing
- Automation workflows
