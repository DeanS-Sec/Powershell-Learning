# Powershell-Learning
Repository for area of "Learn Powershell in a Month of Lunches"

PowerShell Learning
Overview

This repository is a dedicated workspace for learning and practicing PowerShell while following the book Learn PowerShell in a Month of Lunches.

The goal of this repository is to document hands-on progress through PowerShell fundamentals, scripting concepts, automation techniques, and security-focused exercises. Scripts and projects in this repository are intended to support both technical skill development and cybersecurity career growth.

Purpose

This repository serves as:

A structured PowerShell learning environment
A scripting practice workspace
A cybersecurity automation lab
A portfolio of PowerShell progress and projects
Topics Covered

As learning progresses, this repository may include:

PowerShell fundamentals
Variables and objects
Pipelines and formatting
Functions and modules
Error handling
File and system management
Process and service monitoring
Network and port auditing
Windows administration
Security and system auditing
Automation workflows
